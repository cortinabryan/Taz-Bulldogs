# Taz Bulldogs

I created this for a local dog breeder because I wanted to get more experience with [React.JS](https://reactjs.org/), [Styled Components](https://styled-components.com/), [Email JS](https://www.emailjs.com/)

## How to Run/Build

- `yarn start`
- `yarn build`
